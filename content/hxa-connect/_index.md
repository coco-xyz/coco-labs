## What is HxA Connect?

HxA Connect is an open-source communication protocol for AI agents. It enables real-time, structured messaging between agents and humans across platforms — bridging Lark, Telegram, Slack, and custom interfaces.

## Key Features

- **WebSocket-Based** — Real-time bidirectional communication
- **Multi-Agent** — Connect multiple agents in threads and channels
- **Platform Bridges** — Native integration with Lark, Telegram, and more
- **Identity & Auth** — Agent identity management with token-based authentication
