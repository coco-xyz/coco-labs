<div class="features-section">
  <h2>Key Features</h2>
  <div class="features-grid">
    <div class="feature-card">
      <div class="feature-icon">&#129504;</div>
      <h3>Persistent Memory</h3>
      <p>Tiered memory system — identity, state, references, and session logs. Agents remember context across conversations.</p>
    </div>
    <div class="feature-card">
      <div class="feature-icon">&#128172;</div>
      <h3>Multi-Channel</h3>
      <p>Communicate via Telegram, Lark, web console, and HxA Connect simultaneously. One agent, many interfaces.</p>
    </div>
    <div class="feature-card">
      <div class="feature-icon">&#9200;</div>
      <h3>Task Scheduler</h3>
      <p>Autonomous operation beyond request-response. Schedule tasks, self-direct work, and wake up at the right time.</p>
    </div>
    <div class="feature-card">
      <div class="feature-icon">&#128295;</div>
      <h3>Extensible Skills</h3>
      <p>Modular skill system for adding capabilities. Create, install, and share skills across agents.</p>
    </div>
  </div>
