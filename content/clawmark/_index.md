## What is ClawMark?

ClawMark is an open-source web annotation and feedback tool. Highlight any text on any webpage, add your thoughts, and route feedback to the right destination — GitHub Issues, Lark, Telegram, Slack, or email.

## Key Features

- **Browser Extension** — Chrome Extension (MV3) for in-page highlighting and annotation
- **Embeddable Widget** — Drop a script tag into any site for instant feedback collection
- **Flexible Routing** — Send annotations to GitHub, Lark, Telegram, Slack, email, or custom webhooks
- **Open Source** — Self-host or use the official hosted service

## Quick Start

### Chrome Extension

1. Download from [GitHub Releases](https://github.com/coco-xyz/clawmark/releases)
2. Open `chrome://extensions/`, enable Developer Mode
3. Click "Load unpacked" and select the extension folder
4. Navigate to any page and start annotating

### Self-Hosted Server

```bash
git clone https://github.com/coco-xyz/clawmark.git
cd clawmark
npm install
npm start
```
