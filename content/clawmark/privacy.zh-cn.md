# ClawMark 隐私政策

**生效日期：** 2026年3月3日
**最后更新：** 2026年3月3日

ClawMark 是由 COCO（以下简称"我们"）开发的浏览器扩展和网页标注工具。本隐私政策说明 ClawMark 收集哪些数据、如何使用以及您的选择。

## 1. 我们收集的数据

### 标注数据
当您创建标注时，ClawMark 会存储：
- 高亮的文本和您的评论
- 被标注页面的 URL
- 时间戳

### 账户数据（可选）
如果您登录，我们会存储您的显示名称和邮箱地址，仅用于身份验证。

### 页面上下文
ClawMark 读取当前页面 URL 以匹配标注规则并将反馈路由到正确的目标。它**不会**读取或存储完整的页面内容。

## 2. 数据如何使用

- **标注数据**存储在您连接的 ClawMark 服务器上（自托管或官方托管服务），并发送到您配置的目标（如 GitHub Issues、Lark、Telegram、Slack、邮件或 Webhook）。
- **账户数据**仅用于身份验证和标注归属。
- 我们**不会**出售、出租或与第三方共享您的数据用于广告或营销目的。

## 3. 数据存储与安全

- 标注数据存储在服务端数据库中。如果使用官方托管服务，数据存储在安全的云基础设施上。
- 如果您自托管 ClawMark，您对数据存储和保留拥有完全控制权。
- 所有网络通信使用 HTTPS 加密。

## 4. 第三方服务

ClawMark 仅在您**明确配置**标注目标时才与第三方服务集成：
- GitHub（创建 issue/评论）
- Lark / 飞书
- Telegram
- Slack
- 邮件服务
- 自定义 Webhook

除非您设置了路由规则，否则不会向任何第三方发送数据。

## 5. 权限说明

Chrome 扩展请求以下权限：
- **activeTab**：读取当前页面 URL 和选中的文本用于标注
- **storage**：在本地保存您的偏好设置和服务器配置
- **host permissions**：与 ClawMark 服务器通信

## 6. 数据保留与删除

- 您可以随时通过扩展或服务器界面删除单个标注。
- 自托管用户对数据保留拥有完全控制权。
- 如需在官方托管服务上删除您的数据，请通过以下方式联系我们。

## 7. 儿童隐私

ClawMark 不面向 13 岁以下的儿童。我们不会故意收集儿童的数据。

## 8. 政策变更

我们可能会不时更新本隐私政策。变更将发布在本页面，并更新"最后更新"日期。
